Agent.Agent¶

class Agent.Agent.Agent¶

Bases: object

Class representing a radar agent to be run at each radar.

Agent is the top level class for the WARNO Agent microservice. This class is responsible orchestrating the running of the different plugins, handling registration, and communicating data from each plugin up to the event manager.
