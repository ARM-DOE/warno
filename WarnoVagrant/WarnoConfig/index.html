Introduction¶

WarnoConfig is used in Agent, User Portal, and Event Manager for both configuration and utility functions. It provides the sqlalchemy models for the database initialization, helper functions to load in configuration files, enumerators and helper functions used throughout the project, and database helpers to ease loading and initializing the database.
