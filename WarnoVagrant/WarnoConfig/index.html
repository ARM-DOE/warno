WARNO Config¶

Configuration and utility functions to be used throughout the project.

  • WarnoConfig.config
  • WarnoConfig.utility
    • Module Contents

Introduction¶

WarnoConfig is used in Agent, User Portal, and Event Manager for both configuration and utility functions. It provides the sqlalchemy models for the database initialization, helper functions to load in configuration files, enumerators and helper functions used throughout the project, and database helpers to ease loading and initializing the database. Some of these helpers have been deprecated in favor of sqlalchemy builtins.
